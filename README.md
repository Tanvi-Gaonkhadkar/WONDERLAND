# W O N D E R L A N D

### Luxury Vacation Rental & Travel Experience Platform

Wonderland is a full-stack web application where users can explore and book luxurious vacation rental properties, discover beautiful destinations, and share their experiences.
Users can create their own rental entries, upload images, write reviews, and securely manage their listings.

## Features

### User Accounts

Secure signup, login, and logout

Session-based authentication

Only the creator of a rental can update or delete it

 ### Luxury Rental Management

Add new vacation rental properties

Upload high-quality photos through Cloudinary

Edit or remove your own rentals

Each rental includes:

Title

Price

Category (Beach, Mountain, Farm, Lake, etc.)

Description

Location

Photos

### Reviews

Logged-in users can post reviews

Edit or delete your own reviews

Review author information visible


### Responsive Design

Clean, minimal UI

Built with Bootstrap 5 + custom CSS

Fully responsive across devices

## Tech Stack

### Backend

MongoDB + Mongoose

### Frontend

EJS Templating

Bootstrap 5

Custom CSS

### Cloud & APIs

Cloudinary for image hosting
